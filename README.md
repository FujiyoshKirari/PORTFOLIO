 30分

### 操作方法
 ボタンを押しまくれ!!

---PinBall---<br>
https://fujiyoshkirari.github.io/PORTFOLIO/PinBall/
### 制作時間
 5時間
### こだわり
 ピンボールを発射するときの発射装置の動き
### 操作方法
 スペースを長押しして離すとボール発射<br>
 F, Jキーでフリッパーを動かす<br><br>

---Sumo---<br>
https://fujiyoshkirari.github.io/PORTFOLIO/Sumo/
### 制作時間
 5時間
### こだわり
 キャラクターの腕の長さや角度を調節してオリジナルの力士を作れるようにした
### 操作方法
 カスタマイズ画面ではスライダーで力士をカスタマイズ<br>
 バトル中は土俵をクリックして敵を押し出せ!!
