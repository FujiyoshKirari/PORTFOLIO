## ---Sumo---<br>
https://fujiyoshkirari.github.io/PORTFOLIO/Sumo/
### 制作時間
 5時間
### こだわり
 キャラクターの腕の長さや角度を調節してオリジナルの力士を作れるようにしました。
### 操作方法
 カスタマイズ画面ではスライダーで力士をカスタマイズ<br>
 バトル中は土俵をクリックして敵を押し出せ!!

## ---Shooting---<br>
https://fujiyoshkirari.github.io/PORTFOLIO/ShootingGame/
### 制作時間
  7時間
### こだわり
 敵の挙動が体力に応じて変化するようにしました。
 弾は敵もプレイヤーも共通のオブジェクトプールを使用するようにしました。
